### Como executar o projeto

Pré-requisitos: Node JS 14.15.4

```bash
# clonar repositório
git clone https://github.com/jeandossantos/NLW-Heat.git

# entrar na pasta do projeto back end

# instalar dependências
npm i
# executar o projeto
npm start
```
